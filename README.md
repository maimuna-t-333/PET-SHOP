# 🐾 PET SHOP

A clean, responsive landing page designed for a fictional pet store. This project showcases essential frontend development skills using **HTML** and **CSS**, with a focus on layout, design, and usability. It's ideal for learning or demonstrating how to build a simple yet effective user interface.

---


## 🔧 Technologies Used
- **HTML5**
- **CSS3**

---

## ✨ Main Features
- Fully responsive design for desktop and mobile
- Clean, structured layout
- Featured pet products or categories
- Simple call-to-action sections

---

## 📦 Dependencies
No external libraries or frameworks were used.
